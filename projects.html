My Projects

AquaGazen

AquaGazen is an website project focused on demonstrationg a sustainable water distribution powered by biogas. It collects polluted water from various sources, filters it using biogas-powered pumps, and distributes it to clients.

Technologies Used: Django, MySQL, HTML, CSS

Library Management

The Library Management System is a web application designed to facilitate the management of a library's resources. It allows administrators to efficiently manage the library's collection, including adding, updating, retrieving, and deleting books. The application follows the CRUD (Create, Read, Update, Delete) operations to manage books, authors, and categories, ensuring smooth functionality for both library staff and users .

Technologies Used: Flask , MYSQL , HTML, CSS, JavaScript

Image Processing

This Image Processing project leverages Python's powerful libraries, NumPy and Matplotlib, to perform various operations on digital images. The project focuses on applying a range of image processing techniques, such as filtering, transformation, enhancement, and manipulation, to analyze and modify image data for different purposes, like improving visual quality or extracting important features.

Technologies Used: Python , Numpy , Matplotlib
